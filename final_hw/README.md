# lockxuan.github.io/final_hw
  
  
  我的final project是一個貪吃蛇的遊戲，  
  有兩種模式，  
  一、time mode : 比賽計時吃到10分時的秒數  
  二、point mode : 傳統貪吃蛇，吃到高分為主  
    
  遊戲特色 :   
  一、每吃一個速度加快一些  
  二、好玩  
    
      
  ////////////////////////  
  css、js製作特色 :  
  一、hover、字體設計及搭配(使用animate.css、GOOLE FONT)   
  二、以table形式製作整體遊戲  
  三、加入計時豐富有趣(自己寫的)  
  四、除了使用一些library，95%自己寫的  
  五、不變的icon
